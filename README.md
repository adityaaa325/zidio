## Local development

Requirements: Node.js (recommended via nvm) and npm.

Quick start:

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

If you want to deploy, build and serve the `dist/` output with your favorite static host (Netlify, Vercel, etc.).
